## Overview

Predicts Pakistani house prices using a Random Forest model trained on real Zameen.com listings. Built for a Data Analytics Internship at **CodeAlpha**, covering data cleaning, EDA, ML modeling, and AI-powered deployment.

## Features

- 🔮 Instant price prediction with confidence range
- 🏘️ Similar real listings for comparison
- 🤖 Gemini-powered AI assistant for real estate Q&A
- 📈 Interactive data exploration dashboard

## Model Performance

| Model | R² Score |
|---|---|
| Linear Regression | 0.66 |
| **Random Forest** | **0.92** |

Trained on 89,642 cleaned listings after handling outliers, mixed units (Marla/Kanal), and log-transforming skewed price/area distributions.
